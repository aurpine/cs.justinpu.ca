C Style Standard I/O

Intro

C style standard input/output consists of the two methods scanf and printf for input and output respectively.

Both of these methods use format specifiers to indicate what type of variables and format is to be printed or read. The primitive data types all have their own specifier. The format specifiers are contained in a string, similar to how regex works. However, we won't go into much detail about all the features as they are rarely used.

The header for this feature is stdio.h or if you want a better looking name, you can use cstdio with C++.

Format Specifiers

Type Format Specifier
short %h
int %d
long long %ll
char %c
string (char[]) %s
float %f
double %lf
hexadecimal int %x
octal int %o
unsigned int %u

You may combine the last 3 format specifiers with other integral types to read in numbers such as unsigned short (%hu).

Many of these format specifiers however are scarcely used. Just know %d %c %ll %lf.

Application

Now that you know the format specifiers, we can how to use them.

scanf

scanf(format_specifiers, args...);

As demonstrated in the above code, the scanf function has a first parameter as a string containing all the format specifiers. Following, are pointers to the variables to read, as specified in the string. For the simplicity of this lesson, to make a variable into a pointer, add & to the name of the variable. 

int num;
char c;
char[] s;
scanf("%d %c %s", &num, &c, &s);

scanf automatically tokenizes input. This means that the programmer doesn't have to specify what whitespaces to ignore. The spaces between the format specifiers indicate that there will be any amount of whitespace before the next token to be read.

For example, to read an array of N integers you have to read every individual element using a loop. Whether they're separated by spaces, or by new lines, the following code works. 

int N;
scanf("%d", &N);
int numbers[N];
for(int n = 0; n < N; n++)
	scanf("%d", &numbers[n]);

printf

printf works in a very similar aspect. Instead of parsing input, the format specifiers contain the text to be printed, and any format specifiers will be replaced with the given value. 

printf("Hello world!\n");

The above code will print Hello world!. Notice the \n (new line character) at the end. printf doesn't automatically add a new line after it prints.

The method is very concise in the amount of code needed to print. There's no need to concatenate strings using the + operator.

One of the most useful features of printf is that it can format output. There are many format features, but the most used one is printing with specified number precision. 

printf("%.6lf\n", 2.5);

The above code will print 2.500000. The number after the decimal inside the format specifier indicates 6 digits of precision after the decimal point.
