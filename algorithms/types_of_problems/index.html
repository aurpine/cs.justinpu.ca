Types of Problems

There are many types of problems in computer science. Here's a list and a brief explanation of them.

  • Ad hoc - uncategorized, these problems vary from problem to problem. An ad hoc problem may require the use of some other concepts.
  • Simulation - the problem discribed is simulated and a result is printed
  • Implementation - a problem that itself may not be difficult, but implementing a solution may pose as a challenge. Requires usage of data structures.
  • Computational - simple mathematical computation problem, may require some analysis of the problem
  • Brute force - trying every single
  • Recursive - uses recursion to solve the problem like with recursive functions in math, a solution consists of solutions to smaller problems
  • Searches - the problem requires a specific element or value to be found. This category is very broad.
  • Greedy - using short term decisions and assumptions to shortcut to the answer - isn't always right, but fast
  • Dynamic programming - solving and storing previously solved solutions to save time. Works well on recursive problems/solutions.
  • Graph theory - a type of search problem, involving graph nodes/vertices and edges/paths.
  • List may be incomplete - more may be added later on
