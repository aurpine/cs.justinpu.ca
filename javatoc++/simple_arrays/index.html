Simple Arrays

Typically in Java, arrays are declared and initialized as follows: 

int[] a;
int b[]; // Alternative
int[] c = new int[10]; // An int array initialized to size of 5

In C++, the brackets to indicate that the variable is in fact an array, comes after the variable name. It also has to have an integer to indicate its length. This means that you cannot have a global array with a size to be determined in the future. 

int a[5]; // Array of size 5
int b[]; // Error - has to indicate size of array
int[5] c; // Error - [5] has to come after

The above was just declaring int arrays, there are ways to initialize them. The following are all valid methods of initialization. 

int a[5] = {1, 2, 3, 4, 5};
int b[] = {1, 2, 3, 4, 5}; // Compiler assumes size of 5
int c[5] = {1, 2, 3}; // First 3 values have the assigned values
int d[] {1, 2, 3, 4, 5}; // Size of 5
int e[5] {1, 2, 3, 4, 5};
int f[5] {1, 2, 3}; // First 3 have values

2D arrays are very similar. 

int a[2][2];
int b[2][2] = {{1, 2}, {3, 4}};
int c[2][2] {{1, 2}, {3, 4}};

Although you are able to initialize a local array with a variable size, you aren't able to initialize a 2D array with a variable size (nor any other multi-dimensional array). As an alternative, programmers often create global arrays with a very large size-usually the maximum as specified by the input constraints. 

int n = 5;
int a[n]; // Valid
int a[n][n]; // Invalid - error

For example, if the problem states that the maximum length and width of a map to store, you would create a global 2D array with both sizes as 150. The parts of the array that you wouldn't use are quite negligible. Memory is plentiful and there should be enough to solve the biggest test cases, unless of course, your algorithm is wrong. 

int map[150][150];
int main() {
	// Stuff here
}

In Java, there exists the .length method to retrieve the size of the array. Unfortunately, in C++, ways to find the length of an array are complicated. It is way simpler to store the size in a separate variable, since the size is as given from the input, or at least you can keep track of the size, if it were to grow. However, there is an alternate to arrays called vector which will be discussed later. It is the ArrayList of C++. There are many features of the vector class.
