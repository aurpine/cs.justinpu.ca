About the Language

There are many important similarities differences between Java and C++. This lesson is all about these similarities and differences.

Syntax

As you may have noticed in the previous lesson. Code in Java and C++ look relatively the same.

  • ; - the semicolon
  • ? : - ternary operator
  • { } - a block
  • = *= /= += -= %= >>= <<= - assignment operators
  • == <= >= != - boolean operators
  • , - parameter separators
  • etc...

C++ however does have a few additional syntax rules and features. Below is a very basic synopsis of these features and will be discussed in more detail later.

The following code is the C++ equivalent of an import statement 

#include <iostream>

This is an example of pointers and references. 

int a = 5;
int *b = &a;

Files

Java files have the extension .java and are compiled into bytecode with the extension .class

C++ files have the extension .cpp and are compiled into an executable file (.exe).

An executable file can be directly run, while a .class file can not.

Language Mechanics

Java is a multi-platform language and supports many operating systems. This is accomplished with the use of a virtual machine (JVM) on which all Java programs are run on. This may make execution slow.

C++ on the other hand, doesn't support as many languages. Windows and Linux based systems can run them with ease, but other operating systems such as MacOS aren't as compatible. It runs directly from the operating system and is thus faster than Java, but also potentially harmful.
