Math and Calculations

This is another simple type of problem. Often, these problems can be solved using a single calculation or for loop. Math skills can be useful in enhancing the efficiency of an algorithm.

Example 1

Problem: given \(A, B\), the first two angles of a triangle in degrees, find \(C\), the last angle in the triangle.

There are 180° in all the angles of a triangle. Therefore \(A + B + C = 180\). If we balance this equation, we get \(C = 180 - A - B\). We can just code this solution. 

int C = 180 - A - B;

Example 2

Problem: given \(N\), find the sum of all the integers from \(1\) to \(N\) inclusive.

We could solve this problem using a for loop and adding all the numbers together. 

for(int n = 0; n < N; n++) {
    sum += n;
}

There is a better way though. We can use the gauss summation formula, \(\frac{N(N+1)}{2}\), to find the answer with a single calculation. 

int sum = N * (N + 1) / 2;

Example 3

An arithmetic sequence has the terms \(a, a + d, a + 2d, a + 3d ... a + (n - 1)d\). Given \(a\), \(d\) and \(n\), find the \(n\)th term.

One way to do this will be to start off with \(a\) and add \(d\) \(n - 1\) times, using a for loop. 

int term = a;
for(int i = 1; i < n; i++)
    term += i;

Alternatively, we can just see that the \(n\)th term is the last term, which is given in the definition of an arithmetic sequence as \((a + (n - 1)d\). Since we have all three variables, we can find our answer. 

int term = a + (n - 1) * d;
