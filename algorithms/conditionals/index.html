Conditionals

Conditionals are fundamental to programming contests. They are used extensively in other problem types, such as sorting and searching. This lesson will concentrate on simple conditional problems.

If Structure

A common problem is to print output, depending on the input given.

Problem: Print whether a number given is less than, equal to, or greater than the number 10.

To solve this problem, we just use an if structure to check which range it is in. 

get number
if number < 10
	print "less than 10"
else if number > 10
	print "greater than 10"
else
	print "equal to 10"

There are other, more difficult problems requiring the use of conditions.

Switch-Case

Switch blocks are another type of conditionals statements. These aren't often used, but may be faster.

If we were given a number, either 1, 2 or 3, and had to print A, B and C respectively, this will be very easy with a switch-case block. 

switch (num) {
	case 1: print "A"; return;
	case 2: print "B"; return;
	case 3: print "C"; return;
}

A positive feature, although never used, is that after a case is found, it continues running subsequent code.

An example would be if I had 3 types of bags. Bag 1, has an A element. Bag 2 has an A element and a B element. Finally, bag 3 has one of each A, B and C elements. We could then use a switch-case block to find the number of each element there is in a bag. Put this statement in a for loop to find the total number of each element in all the bags. 

switch (bag) {
	case 3: C++;
	case 2: B++;
	case 1: A++;
}

Continuation

Not all conditionals problems are as easy as these examples. As you continue, these problems require you to use compound conditions and math operations within the conditions.
