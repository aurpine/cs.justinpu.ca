Headers

In Java, to expand the library that your program can use, you would import at the top of the file. In C, we include headers.

Headers are all in the following form: 

#include <header>

The header name (inside the tags) must match the header file name exactly. This means no spaces before or after the header name. Notice how there is no semicolon after the line.

The following are a list of a few common headers that you may use.

  • iostream
  • cstdio
  • cmath
  • algorithm

These are the C++ headers. There are the original C headers. For example, cstdio in C is stdio.h. The C++ style headers are preferred for C++.

Trick (bits/stdc++.h)

This is a special header that contains all the headers in the language, so additional includes are necessary. 

#include <bits/stdc++.h>

However, be wary when using this header. It is only supported in the GCC compiler (which you are probably using) and will not work with MS Visual C++ and other compilers.
