Tools and Resources

Here, is a compilation of many great programming resources. IDEs, compilers, websites, judges, etc. Sorted by category.

Contest Sites

These sites often host programming contests. They also have an enormous selection of archived problems that you can attempt to practice. Even better is that you can directly upload your program to be judged.

DMOJ - this is Don Mills CI's programming site. The contests are at our level. Many problems. Supports Turing.

Wcipeg - Woburn CI's programming site. Just like DMOJ, but blue. They often have the same problems.

HackerRank - practice problems are categorized and very easy to use. Intermediate to difficult programming problems.

CodeChef - an Indian website. There are many problems, but may be hard to understand at first.

USACO - the American Olympiad programming site. Ugly and may be hard to understand - but very advanced.

Programming Contests

All the sites listed above do hosts contests, but there are sites that are more specialized for contests.

USACO - there are different contests on varying degrees of difficulty.

DMOJ - monthly contests.

TopCoder - hosts frequent SRMs. These contests are difficult.

Annual Programming Contests (Advanced)

These contests have limited time. You download the input, run your program, then you upload output to be judged.

Facebook Hacker Cup - a decent contest hosted by Facebook. Requires a Facebook account to participate.

Google Code Jam - harder than the Facebook contest.

C++ Documentation

cplusplus.com - the official documentation of C++.

cppreference.com - unofficial reference. May be easier to understand. Looks better.

Java Documentation

Java 7 - the official documentation for Java 7.

Java 8 - the official documentation for Java 8.

Forums

Stack Overflow - search Google and this site will pop up! Many questions and answers. Answers aren't always the best qualify, but usually suffice.

Java IDEs

Eclipse - portable, beautiful, powerful.

Ready to Program - very simple and fast. Java 1.4.2 only.

IntelliJ IDEA - beautiful, powerful IDE.

C++ IDEs

Visual Studio Express - the free version of Visual Studio. Looks great. Uses Visual C++.

Dev-C++ - portable! Does not require a project to run and compile files.

Code::Blocks - free Windows IDE. Multi-compiler support. Cross-platform.

C++ Compilers

Microsoft Visual C++ - comes with Visual Studio.

GCC (GNU Compiler Collection) - linux compiler, needs other software for Windows compatibility.

MinGW - GCC compiler for Windows.

Cygwin - alternate for GCC on Windows.
